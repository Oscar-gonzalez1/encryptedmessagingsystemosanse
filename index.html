Encrypted Messaging System

Project Overview

The project "Encrypted Messaging Systems" aims to develop a secure and efficient system for transmitting predefined messages using Arduino microcontrollers. The information is encrypted to ensure that only the authorized receiver can decipher and understand the message.

The system allows selecting and sending messages via buttons, displaying the information on LCD screens both at the transmitter and the receiver.

RF communication modules and dipole antennas designed specifically for the project are used, ensuring efficient data transmission. The process includes component acquisition, antenna design, system implementation, code development for encryption and visualization, and tests to ensure correct functionality.
